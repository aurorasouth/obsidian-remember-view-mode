# Remember view mode

Remember and restore edit/preview mode per note. When you open a file, it opens in the same mode you left it in.

## Installation

1. Copy `main.js` and `manifest.json` into your vault:  
   `<Vault>/.obsidian/plugins/remember-view-mode/`
2. Reload Obsidian and enable the plugin under **Settings → Community plugins**.

## Settings

- **Remember view mode** — Turn on or off per-note edit/preview memory (on by default).

## Development

From the plugin folder: `npm install`, then `npm run dev` (watch) or `npm run build` (production).
